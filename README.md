# Simple AI Username Generator

Gives you username ideas based on themes or take inspiration from your favorite books, shows, games etc. You just have to type it and let AI do the thinking for you.

---

### Features

- **User Privacy:** Your API key will get saved locally in your web browser.
- **Thematic Search**
  - You can combine two themes `<theme1> + <theme2>`
  - Get Ideas from existing media `Megaman X Boss name ideas`

---

### Tech Stack

- **AI Model:** Gemini 2.5 Flash
- **Language:** Javascript
- **Icons:** Font Awesome

---

### Usage Guide

1.  Get your API key [here](https://ai.google.dev/gemini-api/docs/api-key) from Google.
2.  Copy and paste your API key in the Username Generator (You can also save your API key for your convenience).
3.  Now you can generating.

---

### WiP Features

- Start Generating by simply clicking the Generate button (no more typing specific themes)
- Adding One word and Gibberish type of names.
